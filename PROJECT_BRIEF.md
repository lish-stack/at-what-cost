# Project Brief

## Target User
- 18-34 years old
- Urban, conscious consumers
- Care about corporate ethics
- Want quick, meaningful actions

## Key Screens
1. **Home**: Urgent actions, stats, spotlight
2. **Brands**: Search, filter, scores
3. **Act**: Available actions
4. **Give**: Donation options

## Design System
- **Logo**: <img width="101" height="57" alt="at-what-cost-logo" src="https://github.com/user-attachments/assets/c944502a-9f36-40a7-a209-e340f8f93752" />
- **Font**: Reddit Mono (12-20px; font size + 8px for line spacing)
- **Brand colors**: Burgundy (#660033), Tiber (#0A3345)
- **Accent colors**: Vegan Cream (#FAF7F2), Soft Gray (#E8E8E8)
- Clean, minimal aesthetic
- Google Material Icons Apache 2.0 for familiarity

## Data Needs
- Company database with scores
- Action templates
- Response tracking
- User engagement metrics
